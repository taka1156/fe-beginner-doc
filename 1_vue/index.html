# 1 - 1. Vueを利用したサイト作成

# 環境構築

  • VScodeにVueの拡張機能を入れます。

    • Vetur

  • vue-cliを導入します。

npm install -g @vue/cli
# OR
yarn global add @vue/cli
  • バージョン確認
vue -V

🎉 success

バージョンが返ってきたら成功です。

# プロジェクト作成

vue create [プロジェクト名]
