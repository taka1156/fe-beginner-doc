# 6 - 4 次に何すればいい?

  • vue好き

    • Nuxt.jsでSSGやSSRを試す
    • vue + electronでデスクトップアプリ作成
    • Ionic、Native Script、Weexなどでスマホアプリ作成
      (↑情報が少ないので茨の道になるかも)

  • react好き

    • Next.jsでSSGやSSRを試す
    • React Native
      (↑スマホは勿論、for Mac、Windowsなどもある)

  • sevlte

    • SapperでSSGやSSRを試す
    • sevlte + electronでデスクトップアプリ作成

にそれぞれ進んでみてはいかがでしょうか ヘッドレスCMSと静的サイト生成機能で、ブログなどを作成してみるのも面白いと思います。
