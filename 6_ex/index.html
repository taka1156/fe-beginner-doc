# 6 - 4 次に何すればいい?

  • vue 好き

    • Nuxt.js で SSG や SSR を試す
    • vue + electron でデスクトップアプリ作成
    • Ionic、Native Script、Weex などでスマホアプリ作成
      (↑情報が少ないので茨の道になるかも)

  • react 好き

    • Next.js で SSG や SSR を試す
    • React Native
      (↑スマホは勿論、for Mac、Windows などもある)

  • sevlte

    • Sapper で SSG や SSR を試す
    • sevlte + electron でデスクトップアプリ作成

にそれぞれ進んでみてはいかがでしょうか ヘッドレス CMS と静的サイト生成機能で、ブログなどを作成してみるのも面白いです。
