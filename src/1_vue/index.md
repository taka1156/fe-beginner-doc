# 1 - 1. Vueを利用したサイト作成

## 環境構築
- VScode に Vue の拡張機能を入れます。
  - [Vetur](https://marketplace.visualstudio.com/items?itemName=octref.vetur)

- vue-cli を導入します。

```shell
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

- バージョン確認
```shell
vue -V
```

::: tip 🎉 success
 バージョンが返ってきたら成功です。
:::

## プロジェクト作成

```shell
vue create [プロジェクト名]
```
